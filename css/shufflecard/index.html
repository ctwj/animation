分为3个阶段， 每个阶段区分成不同动画，通过 dealy 控制执行
        1. 初始阶段 正放在中央， 移动到发牌位置
        initMove {
            0% {
                transform: translateY(0px);
            }
            100% {
                transform: translateY(-75px);
            }
        }
        2. 旋转洗牌, 洗完牌，停留在最后状态

        3. 洗完恢复到初始位置
        finishMove {
            100% {
                transform: translateY(0px) translateX(0px);
            }
        }
